# Práce s textovými soubory

Cvičení zaměřené na práci s textovými soubory v C# - čtení, zápis a manipulace s textem.

## Obsah

- Čtení souborů pomocí `File` a `StreamReader`
- Zápis souborů pomocí `File` a `StreamWriter`
- Práce s kódováním (UTF-8, ASCII)
- Počítání znaků a slov
- Odstranění diakritiky
- Analýza četnosti slov a znaků

## Technologie

C# | .NET 8.0
